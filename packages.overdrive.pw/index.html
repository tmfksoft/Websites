The Utopia Repository

View the repo in your browser: dists/ pool/

Welcome to Utopia, James (GLolol)'s personal APT repository for Debian.

At the moment, this repository focuses on importing things not available in stock Debian into one centralized repository, making it easier to install things! Supported architectures are amd64 and i386.

Absolutely no warranty is provided for the use of any software in this repository; use at your own risk!
All credit goes out to the software's original developers.

Instructions

This repository basically works like any other APT repository or Ubuntu PPA, and is signed using a GPG key.

1) Add the repository to sources.list

deb http://packages.overdrive.pw/ sid main import
deb-src http://packages.overdrive.pw/ sid main import

The repository is designed for Debian testing/unstable, but older releases are still compatible to some extent.

Most things in the repository will work with Debian stable and various Ubuntu releases, provided the dependencies are satisfiable. (If they aren't, that means the package isn't compatible with your system! Don't try to forcibly install it!)

2) Install the repository's GPG signing key

wget -O /tmp/sub.class1.server.ca.pem https://www.startssl.com/certs/sub.class1.server.ca.pem
wget --ca-certificate=/tmp/sub.class1.server.ca.pem -O - https://secure.overdrive.pw/debian/repo.gpg.key | sudo apt-key add -

3) Enjoy!

After adding the repositories of your choice, simply run apt-get update to update your repository cache. Then, you will be able to install packages whichever way you like! (apt-get, aptitude, synaptic, etc.)

Components

This repository currently has 2 areas/components:

  • main: Manually (re)packaged packages for this repository (~dimp or ~drepack tag at end), and a few extras I put in.
  • import: The bulk of the goodies: these packages have either been directly imported (mirrored) from an external source, or simply haven't been tested well enough to put in main.

About

Why did I make this?

  1. Simplicity. Instead of adding 5 repositories for the things I commonly use, I only need to add 1.
  2. Ease of use. This repository hosts a few meta-packages (utopia-* in main) to make it easier to pull in a working system. (forget selecting 30 individual packages!)
  3. The learning experience. This is pretty self-explanatory. :)
